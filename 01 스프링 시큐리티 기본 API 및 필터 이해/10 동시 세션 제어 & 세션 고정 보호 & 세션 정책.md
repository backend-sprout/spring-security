# 동시 세션 제어    
동일한 계정으로 인증을 받을 때마다 세션은 새롭게 생성이 된다.     
                
**동시 세션 제어란** 한 계정의 세션 갯수가 일정 기준치를 초과할 경우          
**세션의 갯수를 더 이상 늘리지 않고 유지하는 방법에 대한 제어를 말한다.**
